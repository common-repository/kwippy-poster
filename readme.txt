== Description ==

This is a simple plugin which will post an excerpt of your blog post onto kwippy as soon as you publish.

== Installation ==

1. Upload `kwippypost.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
